# Desafio 17 - Estados do Brasil utilizando Tuplas

## 📖 Sobre

Neste desafio foi desenvolvido um programa para consultar os estados das cinco regiões do Brasil.

O usuário escolhe uma região e o sistema exibe:

- os estados da região;
- o primeiro estado;
- o último estado;
- a quantidade de estados.

Todo o armazenamento foi realizado utilizando **tuplas**.

---

## Funcionalidades

- Menu de seleção das regiões
- Consulta dos estados
- Exibição do primeiro estado
- Exibição do último estado
- Contagem automática utilizando `len()`

## Estrutura do programa

Foram criadas duas funções principais:

```python
titulo()
```

Responsável por exibir o menu.

```python
mostrar_estados()
```

Responsável por exibir todas as informações da região escolhida.
