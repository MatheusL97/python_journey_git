```python

```

## Uso do return

O principal objetivo da aula foi atingido.

O cálculo foi realizado corretamente e o resultado foi devolvido utilizando:

```python

```

## Clareza

# Melhorias futuras

Uma alternativa seria armazenar o retorno em uma variável antes de exibir o resultado.

Exemplo:

```python
resultado = somar(numero_1, numero_2)

print(resultado)
```

Essa abordagem facilita reutilizar o resultado em outras partes do programa.
