# 🚀 Desafio 13 - Calculadora com Return

## 📖 Sobre o desafio

Neste desafio foi criada uma função responsável por somar dois números informados pelo usuário.

A função recebe os valores através de parâmetros, realiza o cálculo e devolve o resultado utilizando `return`.

---

# 🎯 Objetivos

Durante este desafio pratiquei:

- criação de funções;
- parâmetros;
- `return`;
- entrada de dados;
- organização do código.

---

# Funcionalidades

✅ Solicitar dois números ao usuário.

✅ Criar uma função para realizar a soma.

✅ Utilizar `return` para devolver o resultado.

✅ Exibir o resultado no programa principal.

---

# Conceitos utilizados

- Variáveis
- `input()`
- `int()`
- `print()`
- `def`
- Parâmetros
- `return`

---

# Aprendizados

Aprendi que uma função não precisa apenas mostrar informações na tela.

Ela também pode calcular um resultado e devolvê-lo para que o programa utilize esse valor posteriormente.
