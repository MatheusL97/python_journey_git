# 🚀 Desafio 02 — Conversor de Temperatura - 07/07/2026
  
## 🎯 Objetivo

Criar um programa que converta temperaturas entre **Celsius** e **Fahrenheit**.

---

## O desafio

O programa deve perguntar:

```text
Qual conversão deseja fazer?

1 - Celsius para Fahrenheit

2 - Fahrenheit para Celsius
```

Depois perguntar a temperatura.

E exibir o resultado.

---

### Exemplo 1

```text
=== Conversor de Temperatura ===

1 - Celsius para Fahrenheit
2 - Fahrenheit para Celsius

Escolha: 1

Digite a temperatura: 30

Resultado: 86.0 °F
```

---

### Exemplo 2

```text
=== Conversor de Temperatura ===

1 - Celsius para Fahrenheit
2 - Fahrenheit para Celsius

Escolha: 2

Digite a temperatura: 86

Resultado: 30.0 °C
```

---

# 📚 Conceito novo

Hoje você vai aprender algo muito importante:

## 📌 Fórmulas

Para converter Celsius em Fahrenheit:

[
F = (C \times \frac{9}{5}) + 32
]

Para converter Fahrenheit em Celsius:

[
C = (F - 32) \times \frac{5}{9}
]

O desafio aqui não é decorar as fórmulas. É aprender a transformar uma fórmula matemática em código.

# 🎯 Requisitos

Seu programa deve:

* Mostrar um pequeno menu.
* Perguntar a opção.
* Perguntar a temperatura.
* Fazer a conversão correta.
* Mostrar o resultado.

Se o usuário escolher uma opção diferente de **1** ou **2**, exiba uma mensagem informando que a opção é inválida.
