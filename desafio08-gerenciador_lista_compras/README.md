# 🚀 Desafio 08 - Gerenciador de Compras

## 📖 Sobre o desafio

Neste desafio foi desenvolvido um sistema de gerenciamento de compras utilizando listas em Python.

O programa permite adicionar, alterar, remover e visualizar produtos por meio de um menu interativo, consolidando os conhecimentos adquiridos nas aulas sobre listas e estruturas de repetição.

---

# 🎯 Objetivos de aprendizagem

Durante este desafio pratiquei:

- Criação de listas
- Manipulação de elementos por índice
- Adição de itens com `append()`
- Remoção de itens com `pop()`
- Alteração de elementos
- Verificação do tamanho da lista utilizando `len()`
- Estruturas condicionais
- Estrutura de repetição `while`
- Utilização do comando `break`

---

# 📚 Cenário

O objetivo foi criar um sistema simples para controlar uma lista de compras.

O usuário pode:

- Adicionar produtos;
- Alterar produtos existentes;
- Remover produtos;
- Visualizar todos os produtos cadastrados;
- Encerrar o sistema.

---

# 📋 Funcionalidades

### ✅ Adicionar produto

Solicita o nome do produto e adiciona ao final da lista.

---

### ✅ Alterar produto

Solicita a posição do produto e permite alterar seu nome.

---

### ✅ Remover produto

Solicita a posição do produto e remove o item da lista.

---

### ✅ Ver lista

Exibe todos os produtos cadastrados.

Caso não exista nenhum produto, informa que a lista está vazia.

---

### ✅ Sair

Encerra o sistema utilizando o comando `break`.

---
