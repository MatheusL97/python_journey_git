# 🚀 Desafio 12 - Cadastro de Alunos com Parâmetros

## 📖 Sobre o desafio

Neste desafio foi desenvolvido um pequeno sistema de cadastro de alunos utilizando funções com parâmetros.

O objetivo foi compreender como enviar informações para uma função e utilizá-las dentro dela.

---

# 🎯 Objetivos de aprendizagem

Durante este desafio pratiquei:

- criação de funções;
- parâmetros;
- chamada de funções;
- entrada de dados;
- organização de código.

---

# Funcionalidades

✅ Solicitar o nome do aluno

✅ Solicitar a idade

✅ Solicitar o curso

✅ Exibir todas as informações utilizando uma função

---

# Conceitos utilizados

- Variáveis
- `input()`
- `int()`
- `print()`
- `f-string`
- `def`
- Parâmetros

---

# Aprendizados

Aprendi que funções podem receber informações através de parâmetros.

Também compreendi que a coleta dos dados pode ficar no programa principal, enquanto a função fica responsável apenas por executar sua tarefa.
