# 🚀 Desafio 05 - Sistema de Login

## 📖 Sobre o desafio

Neste desafio, o objetivo é desenvolver um sistema simples de login utilizando o laço de repetição `while`.

O programa deverá solicitar a senha ao usuário até que a senha correta seja informada.

Este desafio tem como foco praticar a utilização do `while`, entendendo como repetir uma ação enquanto uma condição for verdadeira.

---

## 📚 Cenário

Imagine que você foi contratado para auxiliar no desenvolvimento de um sistema interno de uma empresa.

Antes de acessar o sistema, o funcionário precisa informar sua senha.

Caso a senha esteja incorreta, o sistema deverá solicitar a senha novamente.

Somente após informar a senha correta o acesso será liberado.

---

Neste desafio pratiquei:

---

## 📋 Regras

A senha correta será:

```text
python123
```

O programa deverá:

- Solicitar a senha ao usuário.
- Continuar solicitando enquanto a senha estiver incorreta.
- Encerrar somente quando a senha correta for digitada.
- Exibir uma mensagem informando que o login foi realizado com sucesso.

---

## 💻 Exemplo de execução

```text
Digite a senha:
123

Senha incorreta.

Digite a senha:
abc

Senha incorreta.

Digite a senha:
python123

Login realizado com sucesso!
```

---

## 🧠 Planejamento (IPO)

### Input (Entrada)

- Senha digitada pelo usuário.

### Process (Processamento)

- Comparar a senha informada com a senha correta.
- Enquanto a senha estiver incorreta, solicitar uma nova tentativa.
- Quando a senha estiver correta, finalizar o programa.

### Output (Saída)

Exibir:

- "Senha incorreta." (quando necessário)
- "Login realizado com sucesso!" (quando a senha correta for informada)
