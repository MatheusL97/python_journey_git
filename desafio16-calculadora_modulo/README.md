# Desafio 15 - Calculadora utilizando Módulos

## 📖 Sobre

Neste desafio foi desenvolvida uma calculadora dividida em dois arquivos Python.

O objetivo foi aprender a reutilizar funções através de módulos utilizando o comando `import`.

---

## Estrutura

```
desafio15/

│

├── calculadora.py

└── main.py
```

---

## Arquivo calculadora.py

Responsável pelas operações matemáticas.

Funções implementadas:

- somar()
- subtrair()
- multiplicar()
- dividir()

Todas retornam o resultado utilizando `return`.

---

## Arquivo main.py

Responsável por:

- solicitar os números;
- apresentar o menu;
- receber a opção do usuário;
- chamar as funções do módulo;
- mostrar o resultado.
