## 🎯 Objetivo

Aprender como um programa toma decisões utilizando estruturas condicionais.

---

## 📖 O que é uma estrutura condicional?

Uma estrutura condicional permite que o programa execute diferentes blocos de código dependendo de uma condição.

Essas decisões são feitas utilizando as palavras-chave:

- if
- elif
- else

---

## 📌 if

O `if` executa um bloco de código apenas se a condição for verdadeira.

Exemplo:

```python
idade = 20

if idade >= 18:
    print("Maior de idade")
````

---

## 📌 else

O `else` é executado quando a condição do `if` for falsa.

Exemplo:

```python
idade = 15

if idade >= 18:
    print("Pode dirigir")
else:
    print("Ainda não pode dirigir")
```

---

## 📌 elif

O `elif` permite testar uma nova condição caso o primeiro `if` seja falso.

Exemplo:

```python
nota = 6

if nota >= 7:
    print("Aprovado")
elif nota >= 5:
    print("Recuperação")
else:
    print("Reprovado")
```

---

## 💡 Como o Python pensa

Toda condição resulta em apenas dois valores:

* True (Verdadeiro)
* False (Falso)

Se a condição for verdadeira, o bloco é executado.

Se for falsa, o Python procura outra condição (`elif`) ou executa o `else`.

---

## ⚠️ Erros comuns

* Esquecer os dois pontos (`:`)
* Esquecer a indentação
* Usar vários `if` quando `elif` seria mais adequado

---

## 🧠 Como pensar

Antes de escrever um `if`, pergunte:

> Qual decisão meu programa precisa tomar?

---

## 📌 Resumo

✔ `if` → primeira condição

✔ `elif` → outra condição

✔ `else` → caso nenhuma condição seja verdadeira
