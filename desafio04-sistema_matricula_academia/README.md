# 🚀 Desafio 04 - Sistema de Matrícula da Academia

## 📖 Sobre o desafio

Neste desafio, o objetivo foi desenvolver um programa capaz de verificar se um aluno está apto a participar de uma turma especial de musculação.

O foco deste exercício foi praticar o uso de **operadores lógicos**, principalmente o operador `and`, combinando duas condições para tomar uma decisão.

---

## 🎯 Objetivos de aprendizagem

Neste desafio pratiquei:

- Entrada de dados com `input()`
- Conversão de tipos utilizando `int()`
- Estruturas condicionais (`if` e `else`)
- Operador lógico `and`
- Variáveis
- Organização e planejamento antes da implementação

---

## 📋 Regras

Para que um aluno possa participar da turma especial, ele deve atender aos seguintes requisitos:

- Ter **16 anos ou mais**;
- Possuir **atestado médico**.

Caso uma das condições não seja atendida, o aluno não poderá participar da turma.

---

## 💻 Exemplo de execução

```text
Digite o nome do aluno:
Matheus

Digite a idade do aluno:
18

O aluno possui atestado médico? (sim/nao):
sim

O aluno Matheus está apto a participar da turma especial de musculação.
```

### Outro exemplo

```text

```

---

## 🧠 Planejamento (IPO)

### Input (Entrada)

- Nome do aluno
- Idade
- Possui atestado médico (`sim` ou `nao`)

### Process (Processamento)

Verificar se:

- A idade é maior ou igual a 16 anos;
- O aluno possui atestado médico.

Caso ambas as condições sejam verdadeiras, o aluno estará apto para a matrícula.

### Output (Saída)

Exibir:

- Nome do aluno
- Situação da matrícula
