# 🚀 Desafio 01 - Calculadora em Python - 06/07/2026

## 📖 Objetivo

Desenvolver uma calculadora simples utilizando apenas os conceitos básicos da linguagem Python.

Ao final deste desafio, o programa deverá ser capaz de realizar as quatro operações matemáticas fundamentais:

- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão

---

## 🎯 Requisitos

O programa deve:

- Solicitar o primeiro número.
- Solicitar o segundo número.
- Perguntar qual operação matemática será realizada.
- Exibir o resultado da operação.

### Exemplo de execução

```text
=== CALCULADORA ===

Digite o primeiro número: 10
Digite o segundo número: 5
Digite a operação (+ - * /): *

Resultado: 50
```

---

## 📚 Conceitos permitidos

Neste desafio, utilize apenas os seguintes recursos da linguagem:

- `print()`
- `input()`
- Variáveis
- `float()`
- `if`
- `elif`
- `else`

---

## 🚫 Restrições

Ainda **não é permitido** utilizar:

- `while`
- `for`
- `match`
- Funções (`def`)

Esses conceitos serão trabalhados nos próximos desafios.

---

## 💡 Dicas

Antes de começar a programar, pense nas seguintes perguntas:

- Quais informações meu programa precisa armazenar?
- Como descubro qual operação o usuário escolheu?
- Como mostrar o resultado de forma clara?
