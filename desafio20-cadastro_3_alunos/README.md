# Desafio 19 - Cadastro de Turma

## 📖 Sobre

Neste desafio foi desenvolvido um sistema simples de cadastro de alunos utilizando listas de dicionários.

Cada aluno é armazenado em um dicionário contendo suas informações.

Todos os dicionários são armazenados dentro de uma lista.

Ao final, o sistema exibe todos os alunos cadastrados e informa a quantidade total de registros.

---

## Funcionalidades

- Cadastro de 3 alunos
- Nome
- Idade
- Curso
- Armazenamento em lista de dicionários
- Exibição organizada dos cadastros
- Contagem dos alunos cadastrados

---

---

## Funcionamento

O programa realiza três etapas:

1. Cadastra os alunos.

2. Armazena cada aluno em um dicionário.

3. Adiciona o dicionário na lista.

Após finalizar os cadastros, todos os alunos são exibidos.

Por último, é mostrada a quantidade de alunos cadastrados.

---

## Exemplo

```
==== ALUNO CADASTRADO ====

Nome: João
Idade: 20
Curso: ADS

==== ALUNO CADASTRADO ====

Nome: Maria
Idade: 22
Curso: Engenharia

Quantidade de alunos cadastrados: 3
```
