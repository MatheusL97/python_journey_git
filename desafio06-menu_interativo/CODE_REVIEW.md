# 💻 Objetivo do desafio

Desenvolver um menu interativo utilizando `while` e `break`, permitindo que o usuário navegue entre as opções até decidir encerrar o programa.

---

# ✅ Pontos positivos

## Organização

O programa ficou organizado em um único laço de repetição, tornando o fluxo de execução simples e fácil de acompanhar.

---

## Uso correto do `while`

Foi utilizada apenas uma estrutura `while`, que controla todo o funcionamento do menu.

Essa é a abordagem mais utilizada para esse tipo de sistema.

---

## Uso correto do `break`

O comando `break` foi utilizado para encerrar o menu somente quando o usuário escolheu a opção **3**.

Isso demonstra boa compreensão do conceito apresentado na aula.

---

## Estruturas condicionais

A utilização de `if`, `elif` e `else` deixou o código organizado e separou claramente cada funcionalidade do sistema.

---

## Legibilidade

O código possui boa organização e nomes simples, facilitando a leitura.

---

# 🔍 Evolução durante o desafio

Na primeira tentativa, a solução foi construída utilizando vários `while`, um para cada opção do menu.

Após a revisão e reflexão sobre o fluxo do programa, a implementação foi reestruturada utilizando apenas um único laço de repetição.

Essa mudança tornou a solução mais limpa, mais simples e mais próxima das boas práticas utilizadas no mercado.

Essa evolução demonstra crescimento no raciocínio lógico e na compreensão do controle de fluxo.
