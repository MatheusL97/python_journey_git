# 🚀 Desafio 11 - Gerenciador de Compras com Funções

## 📖 Sobre o desafio

Neste desafio o objetivo não foi adicionar novas funcionalidades.

O foco foi reorganizar todo o código utilizando funções (`def`), tornando o programa mais limpo, organizado e fácil de manter.

Foi a primeira refatoração realizada durante a Python Journey.

---

# 🎯 Objetivos de aprendizagem

Durante este desafio pratiquei:

- criação de funções;
- chamada de funções;
- organização de código;
- reutilização de código;
- boas práticas de programação.

---

# Estrutura do programa

O sistema foi dividido em funções:

- `mostrar_menu()`
- `adicionar_produto()`
- `alterar_produto()`
- `excluir_produto()`
- `mostrar_lista()`
- `encerrar()`
- `digito_invalido()`

Cada função possui uma única responsabilidade.
