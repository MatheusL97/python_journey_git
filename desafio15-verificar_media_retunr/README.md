# Desafio 14 - Sistema de Avaliação de Alunos

## 📖 Sobre

Neste desafio foi desenvolvido um sistema simples para avaliar a situação de um aluno com base em sua média.

Foram utilizadas funções para separar o cálculo da média da verificação da situação do aluno.

---

## Objetivos

Praticar:

- Funções
- Parâmetros
- Return
- If / Elif / Else
- Organização de código

---

## Funcionalidades

- Informar nome do aluno.
- Informar duas notas.
- Calcular a média.
- Verificar a situação.
- Exibir o resultado.
