# 🚀 Desafio 09 - Exibindo a Lista com `for`

## 📖 Sobre o desafio

Neste desafio foi realizada uma melhoria no sistema de gerenciamento de compras desenvolvido anteriormente.

O objetivo foi substituir a exibição da lista utilizando `print(lista)` pela utilização do laço `for`, tornando a apresentação dos produtos muito mais organizada e agradável para o usuário.

---

# 🎯 Objetivos de aprendizagem

Durante este desafio pratiquei:

- Laço de repetição `for`
- Percorrer listas
- Manipulação de listas
- Estruturas condicionais
- Estrutura de repetição `while`
- Organização de menus
- Exibição formatada de informações

---

# 📚 Cenário

O sistema já permitia adicionar, alterar, remover e visualizar produtos.

Entretanto, a forma de exibição da lista ainda não era amigável.

O objetivo deste desafio foi melhorar essa funcionalidade utilizando o laço `for`.

---

# 📋 Funcionalidades

### ✅ Adicionar produto

Adiciona um novo produto ao final da lista.

---

### ✅ Alterar produto

Permite alterar um produto existente através de sua posição.

---

### ✅ Remover produto

Remove um produto informando sua posição.

---

### ✅ Ver lista

Percorre toda a lista utilizando um laço `for`, exibindo cada produto em uma linha.

Exemplo:

```text
- Arroz
- Feijão
- Café
```

---

### ✅ Sair

Encerra o programa utilizando o comando `break`.

---

# 💻 Exemplo de execução

```text
====== GERENCIADOR DE COMPRAS ======
```
